**Vase mode / spiralize outer contour:** The exported STL is designed to be sliced in
vase mode (spiralize outer contour). Enable this in your slicer.

**Layer height:** Recommended layer height is **0.2 mm**.

**Bottom layers:** Set bottom layers to **0**. If you don't... well then you will not
actually have a wristband xD

**Elephant foot compensation:** Disable elephant foot compensation (or set it to
**0**). Otherwise the first layer of the meander function will be offset.

**X-Y contour compensation:** Set the X-Y contour compensation (sometimes called
"outer wall offset" or "XY compensation") to match your extrusion width / layer
height (e.g. 0.2 mm). This ensures the meander geometry is balanced.

**Speed:** Should be slower than normal to ensure good print quality. Try around 20 mm/s.

**Temperature:** Similar to G-code tips — around 220 °C for TPU, lower for softer materials.
